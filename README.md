Guesser Game                                                                     
============          
Simple Ruby Guessing Game   

Introduction
--------------
Quick and simple project to experiment working with [TCPServer] and [TCPSocket] classes in Ruby.

Installation
--------------

```sh
git clone https://github.com/aleknak/guesser.git guesser
cd guesser/bin
chmod +x guesser client
./guesser &
./client
```
